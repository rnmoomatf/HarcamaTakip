Proje Adı: ParaTakip

Bu projede gelir-giderlerinize ait verileri CouchDB'de tutacak şekilde basit bir web uygulaması geliştirildi. Hem Node.js hem de Flask (Python) tabanlı sunucu yapılarına sahiptir. Kullanıcı arayüzü HTML, CSS ve JavaScript kullanılarak hazırlanmıştır.

Projede yer alan bazı temel özellikler şunlardır:

  • HTML/CSS/JS ile web arayüzü
  • Apache CouchDB ile veritabanı
  • Node.js ile sunucu tarafı
  • Python Flask ile alternatif sunucu
